PYTHON PACKAGES

pyDSA

Python package for Drop shape analysis.

This package allows to measure droplet properties from experimental side-views:

  • Contact angles
  • Contact angle hysteresis
  • Contact line velocities
  • Radius and volume
  • Droplet position and velocity
  • ...
The package offers two ways of detecting the droplet edges (Canny edge detection or contour delimitation) and several ways of fitting the edges to extract the wanted properties (circle, ellipse, spline, polynomial, ...). The possibility to analyze videos allows to study the droplet properties in time.

IMTreatment

Python package to handle and analyze sets of vector and scalar fields.

First purpose of this package is to analyze sets of instantaneous vector fields obtained by PIV (Particle Image velocimetry). Main features include:

  • Import and export from/to various format (text, matlab, vtk, im7, vc7 from Davis Lavision, images) and efficient data storage in compressed binary format (.cimt).
  • Display velocity fields, time profiles, spatial profiles with all representations allowed by matplotlib (quiver, contour, matrix, streamlines, ...).
  • Creation of velocity field resulting from composition of artificial vortex (Burger, Lamb-Oseen, ...) as well as dynamic system of artificial vortices.
  • Vortex detection criterion ($\Gamma_2$, residual vorticity, $\lambda_2$, swirling strength, Q criterion, ...).
  • Various classes and functions to detect and analyze vortex trajectories and properties evolution with time.
  • Basic FTLE (Finite-Time Lyapunov Exponents) implementation that can be used for LCS (Large Coherent Structure) detection.
  • Object-oriented approach make the use of this package simple and straightforward.

Pytisean

Python wrapper around the TISEAN software for non-linear time series analysis.

Allow to use the TISEAN software through python, with the choice of getting results as text files (as TISEAN does) or in Python variables.
